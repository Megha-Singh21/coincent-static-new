Java Programming Fundamentals

  • Basic Syntax: Covers the structure of a Java program including the main() method, class declaration, semicolons, code blocks, and naming conventions. You'll learn how to write, compile, and run your first Java program.
  • Datatypes and Variables: Introduction to primitive types (int, float, boolean, etc.) and reference types (arrays, objects). Covers declaration, initialization, type casting, and variable scope.
  • Conditions: Decision-making using if, else if, else, and switch-case. Understanding logical operators and nested condition structures.
  • Functions (Methods): Creating and using methods. Includes method declaration, parameter passing, return values, method overloading, and recursion basics.
  • Loops: Looping constructs such as for, while, do-while, and enhanced for-each. Includes loop control with break and continue.
  • Data Structures: Use of arrays, ArrayList, HashSet, and HashMap. Includes operations like insertion, deletion, searching, and iterating over collections.
  • OOPs (Object-Oriented Programming): Understanding the pillars of OOP: Classes, Objects, Inheritance, Polymorphism, Abstraction, and Encapsulation. Also includes constructor types and access modifiers.
  • Exception Handling: Techniques to manage runtime errors using try, catch, finally, throw, and throws. Discusses checked vs unchecked exceptions and custom exception classes.
  • Packages: Organizing classes and interfaces into packages. Covers built-in (java.util, java.io) and user-defined packages, and the use of access modifiers.
  • File Handling: Reading from and writing to files using File, Scanner, FileReader, BufferedReader, FileWriter, and PrintWriter. Also includes exception handling in file operations.

Advanced Java Concepts

  • JVM (Java Virtual Machine): Understanding how Java code is executed. Includes JDK vs JRE, class loading, bytecode execution, memory areas, and JIT compiler.
  • Threads: Multithreading concepts, thread lifecycle, creation using Thread and Runnable, thread synchronization, inter-thread communication (wait, notify).
  • Garbage Collection: Memory management using Java's automatic garbage collection. Includes finalize(), reference types, and memory leaks.
  • Generics: Writing reusable and type-safe code using generics in classes, methods, and collections. Covers bounded types and wildcard usage.
  • Streams: Processing collections using Java 8 Stream API. Covers methods like filter(), map(), reduce(), collect(), and usage of parallel streams.
  • Memory Management: JVM memory areas (heap, stack, method area), memory leaks, profiling tools, and garbage collection tuning techniques.
  • Collection Framework: Java’s unified framework for storing and manipulating data. Includes List, Set, Map, and Queue interfaces with classes like ArrayList, HashSet, HashMap.
  • Serialization: Converting Java objects into a byte stream using Serializable. Covers transient keyword and versioning using serialVersionUID.
  • Networking and Sockets: Enabling communication between machines using Java's Socket, ServerSocket, and URL classes. Create basic TCP/IP client-server applications.

Java Ecosystem & Frameworks

  • Build Tools
    • Gradle: A powerful build automation tool using Groovy or Kotlin DSL. Faster and more flexible than Maven. Task-based execution.
    • Maven: A widely-used project management and build tool that uses XML (POM files) to manage dependencies and project configuration.
  • Web Frameworks (Basics)
    • Spring: Core concepts like Inversion of Control (IoC), Dependency Injection (DI), Bean lifecycle, and ApplicationContext.
    • Spring Boot: Build stand-alone Spring applications with minimal setup. Includes auto-configuration, embedded servers, and starter dependencies.
  • ORM (Object Relational Mapping)
    • JPA (Java Persistence API): Standard interface for ORM mapping between Java objects and relational databases using annotations like @Entity, @Id.
    • Spring Data JPA: A Spring-based abstraction over JPA that provides automated query generation and simplifies repository handling.
    • Hibernate: Most popular implementation of JPA, supporting advanced features like lazy/eager loading, caching, and custom query language (HQL).
  • JDBC
    • JDBC Template: Part of Spring JDBC. Simplifies database operations like insert, update, and query execution without boilerplate code.
    • JDBI3: A modern library for working with relational databases using Java. Combines SQL-friendly and annotation-based configuration.
  • Logging
    • Log4J2: A fast and flexible logging framework. Allows logging configuration via XML/JSON/YAML. Supports log levels, appenders, and rolling files.
  • Web Frameworks (Advanced)
    • Spring Core (Advanced): Deep dive into bean scopes, ApplicationContext, lifecycle callbacks, and property injections.
    • Spring Security: Secures Java applications with features like authentication, role-based access, OAuth2, and JWT token integration.
    • Spring Data: Advanced data access patterns, custom query methods, projection, pagination, and auditing.
    • Spring MVC: REST API creation using annotations like @RestController, @GetMapping, @PostMapping, parameter binding, and response formatting.
